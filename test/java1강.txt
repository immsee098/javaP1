16.  해당하는 기본 자료형을 쓰고, 크기(byte)와 기억 가능 범위를 적으시오. 
	(기억 가능 범위는 정수형인 경우만 쓸 것)

	숫자형 
		정수형 
			1) byte / 1바이트 / -128~127
			2) short / 2바이트 / -32868~32768
			3) int / 4바이트 / 
			4) long / 8바이트 / -922경 ~ 922경

		실수형
			1) float / 4바이트
			2) double / 8바이트
	문자형 - char / 2바이트
	논리형 - boolean / 1바이트

17. 다음 세 문장의 출력 결과를 쓰시오.
System.out.println("10+30+70=" + 10 + 30 + 70);   // 10+30+70=103070
System.out.println("10+30+70=" + (10 + 30 + 70)); // 10+30+70=110
System.out.println(10 + 30 + 70+" = 10+30+70");   //110=10+30+70
